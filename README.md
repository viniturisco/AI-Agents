# AI Agents

Este repositório contém os arquivos de agentes .md que criei para utilizar junto no vscode, pelo GitHub Copilot. Eles foram criados após o mini curso de IA dentro do bootcamp de Node.js da DIO. Valeu Felipão da DIO pelo curso que tem sido excelente até aqui!

## Agentes que utilizo:

Tenho utilizado quatro agentes até agora

* **Cortana (modo Agent):** Para ser minha agente que ajuda fazendo mudanças no código, arrumando pequenos trechos ou desenvolvendo features que pedi.
* **Gekko (modo Study):** Baseado no personagem de Valorant, Gekko é um agente amigável que me ajuda a estudar conceitos de programação.
* **Jarvis (modo Ask):** Tem a inspiração na famosa IA do bilionário Tony Stark, que me ajuda respondendo perguntas sobre programação, conceitos de código e desenvolvimento.
* **Richie (modo Plan):** Richie é o melhor amigo do herói Super-Choque, e me ajuda a planejar aplicações antes de começar a fazê-las.
